Curricula

Full Stack React App

This is a project I worked on with a team of junior devs to create a proof of concept app. Curricula is a project designed for business level users to be able to quickly visualize what high level skills are taught in which curricula we offer at Revature. Curricula allows for building new curricula and attaching high level skills to them. It puts high level skills in related categories for easy management. Curricula can also build compare contrast visualizations between multiple different curricula.

This site was built with React, Spring, and Postgres
